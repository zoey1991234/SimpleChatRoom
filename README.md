# Simple-ChatRoom
 
# 概要

初學者練習基於 ASP.NET Core 和 SignalR 的聊天板簡易頁面

# 環境配置

開發環境：Windows 10　，　Visual Studio 2022

# 專案結構
## 前端 
HTML、JavaScript、Tailwind CSS

## 後端 
ASP.NET Core：用於建構 Web API 和 SignalR 服務器端 

SignalR：用於實現實時通信功能

---------------------------------------
# 主頁面

用户可以輸入名字和消息 ，消息會顯示給所有連接的用戶
